# Legal Docs Workspace

本仓库用于统一托管多个 App 的法务与支持页面。

## 目录约定

- `booktrail/`：BookTrail 相关页面
- `byesheep/`：ByeSheep 相关页面

每个应用目录建议至少包含：

- `privacy.html`
- `terms.html`
- `support.html`

## 线上访问

- 根入口：`https://legal.mingosoly.com/`
- BookTrail 隐私：`https://legal.mingosoly.com/booktrail/privacy.html`
- BookTrail 条款：`https://legal.mingosoly.com/booktrail/terms.html`
- BookTrail 支持：`https://legal.mingosoly.com/booktrail/support.html`
